PRUEBAS DE HIPÓTESIS

Las pruebas de hipótesis son el pilar de la estadística inferencial. Permiten tomar decisiones sobre un parámetro poblacional (como la media o la proporción) basándose en la evidencia de una muestra, con un riesgo conocido de error.

Concepto clave en la toma de decisiones estadísticas

Tipos y Proceso General

El proceso de decisión estadística se basa en cinco pasos fundamentales. La hipótesis nula (\(H_0\)) es la afirmación que se asume verdadera y se intenta refutar:

$$H_0: \text{El parámetro es igual al valor hipotético}$$

Se compara el valor p con el nivel de significancia (\(\alpha\)), generalmente 0.05.

Pruebas Paramétricas

  • Asumen distribución Normal de los datos.
  • Usan parámetros como la media (\(\mu\)) y la varianza (\(\sigma^2\)).
  • Ejemplos: Prueba Z, Prueba T, ANOVA.

Pruebas No Paramétricas

  • No requieren suponer una distribución específica (distribución libre).
  • Usan rangos, medianas y frecuencias. Ideales para datos ordinales.
  • Ejemplos: Mann-Whitney U, Wilcoxon, Chi-cuadrado (\(\chi^2\)).

Ejemplos Clave

Una fábrica afirma que el peso promedio de sus botellas es de 500 gramos. Se toma una muestra de \(n=35\) botellas, dando una media muestral (\(\bar{x}\)) de 495 gramos. Se sabe que la desviación estándar poblacional (\(\sigma\)) es de 10 gramos. ¿Hay evidencia de que el peso real es diferente a 500g? (\(\alpha = 0.05\))

$$H_0: \mu = 500 \quad H_1: \mu \neq 500$$

Paso 1: Identificar datos y fórmula

Se utiliza la Prueba Z para la media, ya que la desviación estándar poblacional (\(\sigma\)) es conocida y \(n \ge 30\):

$$Z = \frac{\bar{x} - \mu_0}{\frac{\sigma}{\sqrt{n}}} = \frac{495 - 500}{\frac{10}{\sqrt{35}}}$$

Paso 2: Calcular el estadístico de prueba

El error estándar es \(\frac{10}{\sqrt{35}} \approx 1.69\):

$$Z = \frac{-5}{1.69} \approx -2.96$$

Paso 3: Conclusión

El valor crítico para \(\alpha=0.05\) (bilateral) es \(\pm 1.96\). Como \(-2.96 < -1.96\), se rechaza \(H_0\):

$$\text{Se rechaza } H_0$$

Uso del Z-Test (Paramétrico)

Este test se aplica cuando tenemos una muestra grande (\(n \ge 30\)) o cuando conocemos la desviación estándar poblacional (\(\sigma\)). Si \(\sigma\) fuera desconocida y \(n\) pequeña, se usaría la prueba T.

  • **Rechazo de H₀:** El valor $Z$ calculado (-2.96) cae en la región de rechazo (más allá de -1.96).
  • **Significado:** Existe evidencia estadística suficiente para concluir que el peso promedio de las botellas no es 500 gramos.

Una encuesta reciente afirmó que el 60% de los votantes apoya al candidato A. En un nuevo sondeo de \(n=200\) personas, 130 manifestaron su apoyo. Pruebe si la proporción de apoyo es *mayor* al 60% (\(\alpha = 0.05\)).

$$H_0: p \le 0.60 \quad H_1: p > 0.60$$

Proporción muestral: \(\hat{p} = 130/200 = 0.65\).

Paso 1: Aplicar la fórmula de Proporciones

Se utiliza la fórmula Z para la proporción (\(\hat{p}=0.65\), \(p_0=0.60\), \(n=200\)):

$$Z = \frac{\hat{p} - p_0}{\sqrt{\frac{p_0(1-p_0)}{n}}} = \frac{0.65 - 0.60}{\sqrt{\frac{0.60(0.40)}{200}}}$$

Paso 2: Calcular el estadístico de prueba

El error estándar es \(\sqrt{\frac{0.24}{200}} \approx 0.0346\):

$$Z = \frac{0.05}{0.0346} \approx 1.44$$

Paso 3: Conclusión

El valor crítico para \(\alpha=0.05\) (cola derecha) es \(Z_{crítico} = 1.645\). Como \(1.44 < 1.645\), no se rechaza \(H_0\):

$$\text{No se rechaza } H_0$$

Interpretación del Z-Proporción

La prueba Z para proporciones se basa en la aproximación de la distribución binomial a la distribución normal, siempre y cuando \(n \cdot p_0 \ge 10\) y \(n \cdot (1-p_0) \ge 10\).

  1. El valor $\hat{p} = 0.65$ es mayor que $p_0 = 0.60$, pero la diferencia no es lo suficientemente grande para ser estadísticamente significativa.
  2. No hay evidencia suficiente para concluir que el apoyo al candidato A sea mayor al 60%.

Fórmulas Esenciales

Prueba Z para la Media ($\sigma$ conocida)

$$\large Z = \frac{\bar{x} - \mu_0}{\frac{\sigma}{\sqrt{n}}}$$

Donde $\bar{x}$ es la media muestral y $\sigma$ la desviación poblacional.

Prueba T para la Media ($\sigma$ desconocida)

$$\large T = \frac{\bar{x} - \mu_0}{\frac{s}{\sqrt{n}}}$$

Donde $s$ es la desviación muestral y $GL = n-1$ grados de libertad.

Prueba Z para la Proporción

$$\large Z = \frac{\hat{p} - p_0}{\sqrt{\frac{p_0(1-p_0)}{n}}}$$

Donde $\hat{p}$ es la proporción muestral y $p_0$ la proporción hipotética.

Prueba Chi-Cuadrado (No Paramétrica)

$$\large \chi^2 = \sum \frac{(O_i - E_i)^2}{E_i}$$

Donde $O_i$ son frecuencias observadas y $E_i$ las esperadas.
