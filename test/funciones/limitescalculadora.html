Función y Límite

Instrucciones

  • Ingrese una función matemática válida.
  • Ingrese el valor al que $x$ se aproxima.
  • Use el teclado matemático para símbolos especiales.
  • Presione "Calcular Límite" para ver resultados.
  • Use "Limpiar" para borrar la entrada y la gráfica.
  • "Ir a la Teoría" lo llevará a una página de recursos.
  • Funciones soportadas: `sin()`, `cos()`, `tan()`, `sec()`, `csc()`, `cot()`, `ln()`, `exp()`, `sqrt()`, `abs()`, `log()`.
  • Para infinito, use `inf` o `\\infty`.
